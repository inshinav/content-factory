# Слоты под реальные изображения

Положите сюда файлы — компоненты подхватят их автоматически (иначе показываются
намеренные градиентные плейсхолдеры):

- `persona-1.jpg` … `persona-4.jpg` — аватары персонажей (9:16 или квадрат)
- `clip-1.jpg` … `clip-6.jpg` — кадры роликов для TikTok-мокапов (9:16)

Имена задаются в `src/data/personas.ts` (поле avatar/clip).
