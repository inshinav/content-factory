# Контент-завод

Design-first одностраничный **концепт-демо**: визуально показывает автоматизированную
фабрику AI-инфлюенсеров для TikTok — от брифа до тысяч роликов на множестве аккаунтов
и гео. Бренд TikTok-нативный: глубокий чёрный, неон cyan/magenta, чертёжная сетка цеха,
индустриальная кинетика конвейера.

> **Это концепт.** Персонажи вымышлены и brand-safe (lifestyle / wellness / fitness /
> travel). Цифры подписчиков, охвата и масштаба — иллюстративны, не факт. Мокап
> интерфейса TikTok стилизованный, без копирования проприетарного лого.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173 (или порт из launch.json)
npm run build    # tsc + vite, deploy-ready (Vercel)
npm run lint
```

## Стек

Vite · React 18 · TypeScript (strict) · Tailwind v4 (`@tailwindcss/vite`, токены —
CSS-переменными) · motion v12 (`motion/react`) · lucide-react. Шрифты самохостятся
(fontsource, с кириллицей): **Unbounded** (дисплей), **Inter** (body),
**JetBrains Mono** (данные/счётчики).

## Структура

- `src/App.tsx` — композиция секций (конвейер → лента → блогеры → масштаб → машинерия → закрытие).
- `src/components/` — секции и две **сигнатуры**: `TikTokPhone.tsx` (UI-точный мокап:
  табы, action-rail, музыка, таб-бар, прогресс) и `ConveyorSection.tsx`
  (анимированный сборочный конвейер с едущими единицами, свечением станций и петлёй
  обратной связи; вертикальный на мобиле).
- `src/data/` — весь контент: `pipeline.ts` (станции + машинерия), `personas.ts`
  (персонажи), `scale.ts` (счётчики масштаба + гео).
- `src/index.css` — токены и база (тёмные + неон, чертёжная сетка). `src/styles.css` — компоненты.

## Реальные изображения и тексты

Плейсхолдеры намеренные (градиентные аватары, motion-кадры). Чтобы подставить реальные:

- положите файлы в `public/assets/` — `persona-1.jpg`…`persona-4.jpg` (аватары),
  `clip-1.jpg`…`clip-4.jpg` (кадры роликов 9:16);
- имена/хэндлы/ниши/цифры персонажей — в `src/data/personas.ts`;
- счётчики масштаба — в `src/data/scale.ts`.

Если файла нет — показывается намеренный плейсхолдер, не «битая» картинка.

## Доступность и движение

Тёмная тема, контраст AA (моно/мелкий текст проверены на `#0A0A0B`). Только нативные
`button`/`a`, видимый фокус. Все бесконечные циклы (конвейер, муздиск, бегущая строка,
пульс станций и гео) глушатся при `prefers-reduced-motion` — остаётся статичный кадр,
контент виден. Анимации — только `transform/opacity/filter`. Без localStorage.
